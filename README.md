## 数据库设计

```

user=====用户表
 |-------id:编号
 |-------username:用户名
 |-------password:密码
 |-------name:名字
server===服务器列表
 |-------id:编号
 |-------ip:主机IP
 |-------name:主机名
 |-------brand:品牌
 |-------model:型号
 |-------sn:序列号
 |-------location:机架位置
 |-------idc_name:IDC名称
 |-------system:系统环境
 idc=====机房列表
 |-------id:编号
 |-------type:类型
 |-------ips:公网IP地址
 |-------address:机房地址
 |-------contact:联系方式
 host====主机列表
 |-------id:编号
 |-------name:主机名
 |-------ip:IP地址
 |-------group:主机组
 |-------app:功能说明
 |-------status:使用状态
 |-------server_name:所属服务器
 group===主机组信息
 |-------id:编号
 |-------name:主机组名称
 host_detail===主机详细信息
 |-------id:编号
 |-------cpu_model: cpu型号
 |-------cpu_nums: cpu个数
 |-------mem: 内存大小
 |-------disk: 磁盘大小
 |-------hostname:主机名
 |-------ip: ip地址
 |-------ip2: 其他ip地址
 |-------os:操作系统
 |-------service_tag: 序列号


```
